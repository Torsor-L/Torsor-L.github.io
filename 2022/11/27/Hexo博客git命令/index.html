Hexo博客git命令

1、使用Hexo写博客

Hexo博客引擎写博客，可分作以下几步：

  • 建立新的文章文件
  • Hexo博客编译

1、建立新的文章文件

在Hexo的博客目录下的source\_posts文件夹内新建一个文本文档，重命名为xxxx.md格式，然后双击打开在编辑器中写好文章内容

如果你不了解Markdown语言，请花几分钟了解一下Markdown语法教程

2、Hexo博客编译

写完博文xxxx.md后，回到Hexo博客目录，右键，Git Bash Here，进入git编译界面


2.1 编译文件

编译界面输入hexo g，回车，即可编译完成，如果出现报错，可以尝试依次输入hexo clean、hexo g，如果仍然出错，则一般是Hexo设置出错，需要重新检查设置。

2.2 生成本地静态预览页面

输入hexo s，生成本地预览界面http://localhost:4000/，右键copy到浏览器地址栏即可预览本地静态页面，按下Ctrl+C退出预览界面

注意此时仅仅是本地静态预览，只有该电脑上才可以看到，地址发给别人是看不到内容的

2.3 部署到远程(Github Page或你的个人域名)

退出静态预览后，输入hexo d，即可将更改部署到远程服务器，部署成功后等待几分钟就可以在你的个人博客上看到博客内容更新了！

3、附

到此为止一般就大功告成了，对于用Hexo发布博客，Hexo官方文档也有详细说明，建议搭配官方文档学习~

这是我严格意义上的第一篇个人博客，git命令还不太熟练，权当先给自己做个笔记备忘:sunglasses:。下一期也许做Markdown常见语法汇总，也可能直接记一些课堂笔记。:new_moon_with_face::full_moon_with_face:
